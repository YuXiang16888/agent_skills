# agent_skills

專門用來存放和管理 AI 助手技能（Skills）的集中式儲存庫。

## 📋 專案簡介

agent_skills 是一個用於收集、整理和分享各種 AI 助手技能的專案

## 🎯 專案目的

- 集中管理不同 AI 平台的技能定義
- 提供可重複使用的技能模板
- 促進 AI 技能的分享與協作
- 標準化技能格式與最佳實踐

## 📁 目錄結構

```
agent_skills/
├── xxxxxxxx_skill   # xxxxxxxx技能
└── README.md        # 專案說明文件
```

## 🚀 快速開始

1. 選擇你需要的技能
2. 根據技能說明文件進行配置
3. 將技能導入到你的 AI 助手中

## 📝 技能格式

每個技能應包含：

- **名稱**：技能的唯一識別名稱
- **描述**：技能的功能說明
- **使用說明**：如何使用該技能
- **範例**：實際應用範例
- **相依性**：所需的其他技能或工具

## 🤝 貢獻指南

歡迎提交新的技能或改進現有技能！

1. Fork 本專案
2. 創建你的功能分支
3. 提交你的變更
4. 推送到分支
5. 開啟 Pull Request
